## 内容农场

Content Farm，是指为了谋取广告费等商业利益，或出于控制舆论、带风向等特殊目的，快速生产大量网络文章来吸引流量的网站。

特征：

1. 利用 API 和 RSS 对别人博客内容采集后产生的内容
2. 通过关键词和正常文章（多以小说为主）组成的乱七八糟的文章
3. 批量产生的大量质量差的信息内容
4. 复制别人的原创内容
